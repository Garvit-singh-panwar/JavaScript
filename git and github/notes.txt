🧠 Git & GitHub – Quick Notes
🔹 What is Git?
Git is a free & open-source version control system
It helps to track changes in code

👉 Simple meaning:
Git = tool that remembers every change in your code

🔹 Why we use Git?
✅ 1. Track Changes
Saves code step by step (commits)
You can see:
what changed
when changed
✅ 2. Go Back to Old Code
If new code breaks ❌
You can go back to old version ✅

👉 No need to delete code manually

✅ 3. Work Safely
Your old code is never lost
You can experiment without fear
✅ 4. Team Work (Collaboration)
Many developers can work together
Git tracks:
who wrote code
what changes were made

👉 Helps manage conflicts (same file edits)

✅ 5. Fast & Scalable
Works well even for big projects
🔹 Important Terms
Repository (Repo) → Project folder
Commit → Saved version of code
Branch → Separate line of work
Merge → Combine code
🔹 What is GitHub ?
GitHub is a website to store your Git projects online

👉 Simple meaning:
GitHub = online storage for your code

🔹 Git vs GitHub
Git	GitHub
Tool (software)	Website
Tracks code changes	Stores code online
Works offline	Needs internet
🔹 Easy Example
Git = 📓 Notebook (tracks your work)
GitHub = ☁️ Cloud storage (stores your notebook online)
🔹 One-Line Revision

👉 Git = tracks code changes
👉 GitHub = stores code online
