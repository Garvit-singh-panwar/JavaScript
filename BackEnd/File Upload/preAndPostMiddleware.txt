Pre and Post middlewares


the post middlewares work is after creating the entry your middleware will invoke let say i have to do some work before creating a entry then i can use a pre middleware instead of post if i want to do some work after creating a entry in DB i use post middleware 

POST middleware are executed after the hooked method and all its pre middleware have completed 
Pre middleware are excuted before the hooked method 


we use it in sending mail after creating a entry in database

DB-entry------>sending email
