     JWT :  JSON web token 

     JWT is a secured way to transfer information  
     whenever you login to a web application you server generates a  JWT token 
     it is made up of three things 
     1. header      ----->   define         --->    hashing algorithm or (type -> JWT) 
     2. payload     ----->   contain data   --->    userId , userName , userRole
     3. signature   ----->   contain key    --->    secret which identify you any string

    it generates a Json web token (JWT) which contain all this data 
    this token is given to clint by your server 
    clint use this token again and again in different different areas in that application so does'nt need to login again and again 
    he just have to login one time
    clint use this jwt token in eveery req  to prove his authenticity
